Para la ejecución* del código hay dos opciones:
Opción 1 - Descargar el archivo:
1. Descargar el archivo de extensión .java de preferencia en el disco C y guardar la ubicación ejm: C:\Users\andre\Downloads\AppEstadoEnlace.java
2. Abrir el command prompt (o ventana de consola) del sistema e ingresar:
2.1 Ubicación de la carpeta donde se guardó el archivo, siguiendo el ejemplo: C:\Users\andre\Downloads
2.2 Ingresar el comando: javac AppEstadoEnlace.java -> Para compilar el código
2.3 Ingresar el comando: java AppEstadoEnlace.java -> Para ejecutar el código

Opción 2 - Copiar en bloc de notas:
1. Crear un archivo de bloc de notas y copiar el código del repositorio
2. Guardar el archivo como NombreDeseado.java en la ubicación que se desee y guardar la unicación.
3. Abrir el command prompt (o ventana de consola) del sistema e ingresar:
3.1 Ubicación de la carpeta donde se guardó el archivo, siguiendo el ejemplo: C:\Users\andre\Downloads
3.2 Ingresar el comando: javac AppEstadoEnlace.java -> Para compilar el código
3.3 Ingresar el comando: java AppEstadoEnlace.java -> Para ejecutar el código

*Es necesario tener instalada y activada la última versión de Java (15) en el ordenador
